Book Downloader

Steps to Download and Decrypt

1. Click the following button.

2. The ACSM file will be downloaded automatically.

3. Install Adobe Digital Editions and open the ACSM file.

4. It will download an encrypted PDF to the "Documents/My Digital Editions" folder.

5. Install Calibre and the NoDRM plugin from github.com/noDRM.

6. Drag the PDF into Calibre to remove the DRM.
